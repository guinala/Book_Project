Rule: react-doctor/click-events-have-key-events
Severity: warning
Category: Accessibility
Count: 9

Visible non-interactive elements with click handlers must have a corresponding keyboard listener (`onKeyUp`, `onKeyDown`, or `onKeyPress`).

Suggestion: Pair `onClick` with `onKeyUp` / `onKeyDown` / `onKeyPress` for keyboard users.
